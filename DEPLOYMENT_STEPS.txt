خطوات رفع وتشغيل مشروع Flask على cPanel:

1. ضغط المشروع:
   - اضغط كل ملفات ومجلدات المشروع في ملف zip.

2. رفع المشروع:
   - ادخل إلى cPanel > File Manager.
   - ارفع ملف zip إلى public_html أو مجلد فرعي.

3. فك الضغط:
   - بعد الرفع، اضغط على الملف واختر Extract.

4. إعداد بيئة Python:
   - من cPanel ابحث عن "Setup Python App" أو "تطبيقات Python".
   - أنشئ تطبيق جديد (اختر Python 3.8 أو أعلى).
   - اختر المسار إلى مجلد المشروع (مثال: /home/username/public_html/yourproject).
   - حدد اسم ملف التطبيق: wsgi.py

5. تثبيت المتطلبات:
   - من إعدادات التطبيق اضغط "Enter to the virtual environment".
   - افتح Terminal أو استخدم "Start Terminal" من cPanel.
   - نفذ:
     pip install -r requirements.txt

6. ضبط متغيرات البيئة:
   - أضف متغيرات ملف .env في إعدادات البيئة في cPanel أو أنشئ ملف .env في نفس مجلد المشروع.

7. التأكد من وجود ملف wsgi.py بهذا المحتوى:
   from app import app as application

8. تشغيل التطبيق:
   - من "Setup Python App" اضغط Restart أو Update.

9. ربط الويب هوك:
   - استخدم رابط السيرفر (مثلاً: https://yourdomain.com/webhook) في إعدادات فيسبوك.

ملاحظات:
- إذا احتجت لرفع ملفات وسائط، تأكد أن مجلد static/ لديه صلاحيات الكتابة.
- إذا ظهرت أخطاء، راجع سجل الأخطاء في cPanel أو ملف error_log.
- بعض شركات الاستضافة لا تدعم بعض مكتبات الذكاء الاصطناعي، استخدم VPS إذا لزم الأمر.
